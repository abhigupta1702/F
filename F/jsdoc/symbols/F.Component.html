Generic component class

Component options
{Boolean} options.singly
Whether this component will allow multiple sub-components to be visible at once. If true, only one component will be visible at a time.
{Boolean} options.visible
If true, this component will be visible immediately.
{Boolean} options.debug
If true, show debug messages for this component.

{Object} applyOptions(options)
Applies passed options to instance options and applies instance options to passed options. Individual passed options will not be applied to instance options unless they are defined in default options for the class or parent class.
Note: The options merge is one level deep.
Note: This function assumes that this.options does not refer to an object in the prototype.
